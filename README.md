# API de Ternurines (Sylvanian Families) v1.0.0

Bienvenido al repositorio de la API de Ternurines. Esta API proporciona una manera sencilla de gestionar y obtener información sobre ternurines, pequeñas criaturas adorables.

## Características

### Lista de ternurines

Obtén una lista completa de todos los ternurines registrados.

### Detalles de ternurines

 Accede a la información detallada de cada ternurine por su ID.

### Gestión de ternurines

Añade, actualiza y elimina ternurines de la base de datos.

## Uso

### Clonar repositorio
   ```bash
   git clone https://github.com/nayaracasz/ternurines-API.git
   ```

Una vez que el servidor esté en funcionamiento, puedes interactuar con la API utilizando herramientas como curl, Postman o cualquier cliente HTTP.

Si se requiere acceder a la documentación, se encuentra en la ruta del servidor **/api-docs**.

**Servidor local** - http://localhost:3000/
